# EXPELLED — Set 01 : Nés du silence

**Nés du silence** se déroule juste après la Fracture.  
KOR vient de se taire. Le Vasis prononce les premières sentences. Les peuples deviennent ce qu'ils
sont sous la pression de l'Expulsion.

Le set raconte des premières fois :

- premier halo brisé ;
- première chaîne d'or ;
- premier Exar relevé dans Exen ;
- premier Morar heureux de tomber ;
- premier humain dont un mot porte.

---

## 3. Vocabulaire de jeu

| Terme | Fonction |
|---|---|
| **Volonté** | Ressource principale, issue de NER. |
| **Intégrité** | Points de vie d'un être ou d'un joueur, issue de XEN. |
| **Être** | Carte unité : personnages, créatures. |
| **Verbe** | Carte action, sort ou effet ponctuel. |
| **Relique** | Carte objet : permanent équipable ou posé. |
| **Lieu** | Carte domaine : permanent global, un seul Lieu actif par camp. |
| **Prononcer (n)** | Effet puissant payé en Volonté, puis exil définitif de la carte. |
| **Thaen** | Champ de duel, nom possible du plateau ou de l'arène. |
| **Korum** | Nom possible du noyau joueur, de la relique centrale ou de l'objectif à protéger. |

Mots-clés v1 retenus :

- **Serment** : l'Être doit être attaqué avant le Korum tant qu'il est prêt.
- **Élan** : l'Être peut attaquer le tour où il arrive en jeu.
- **Mémoire** : regarder, récupérer ou copier une carte de la défausse ou de l'exil.
- **Rupture** : effet déclenché quand une carte est détruite, sacrifiée ou exilée.
- **Mutation** : choix de modes ou modification temporaire de rôle/statistiques.

## 4. Identités de gameplay

### Vasar

Plan de jeu : stabiliser, protéger, réciter.

Forces :

- buffs de masse ;
- protection ;
- renvoi en main ;
- punition des `Prononcer` adverses ;
- maintien de plateau.

Faiblesses :

- lent ;
- dépendant de l'ordre établi ;
- vulnérable quand la structure tombe.

### Exar

Plan de jeu : casser, sacrifier, finir.

Forces :

- destruction ciblée ;
- effets `Prononcer` très puissants ;
- sacrifices rentables ;
- pression directe sur l'Intégrité.

Faiblesses :

- se blesse soi-même ;
- consomme ses ressources ;
- peut s'épuiser trop vite.

### Eshar

Plan de jeu : savoir, piocher, récupérer.

Forces :

- pioche ;
- manipulation de main ;
- information ;
- interactions limitées avec l'exil.

Faiblesses :

- corps faibles ;
- tempo lent ;
- dépendance aux synergies.

### Morar

Plan de jeu : s'adapter.

Forces :

- changement de rôle ;
- coûts réduits ;
- flexibilité ;
- réponses contextuelles.

Faiblesses :

- moins explosif ;
- moins spécialisé ;
- demande de bien lire la partie.

### Velar

Plan de jeu : brûler vite.

Forces :

- Élan ;
- dégâts directs ;
- pression rapide ;
- pics de Volonté.

Faiblesses :

- faible en partie longue ;
- défense limitée ;
- main qui se vide vite.

## 5. Barème provisoire

Les chiffres hérités des tests précédents restent utilisables comme base, mais doivent être
renommés et recalibrés pour Expelled.

### Unités

Pour une commune :

```text
ATQ + Intégrité ≈ 2 × coût
```

Tolérances :

- rare : léger bonus si l'effet est situationnel ;
- épique : effet plus structurant ;
- légendaire/prismatique : +2 points de marge maximum.

### Effets

Repères :

- pioche 1 carte : 2 points ;
- infliger X dégâts : X points ;
- zone/AoE : environ 2,5 × X ;
- mot-clé simple : 1 point ;
- protection forte : à tester au cas par cas.

### Prononcer

`Prononcer (n)` peut dépasser un effet normal :

```text
valeur d'effet ≤ 2 × n
```

Justification : l'effet est unique et retire la carte du duel.

## 8. Cartes prioritaires à créer

Pour valider le set, produire d'abord :

1. une commune Vasar simple ;
2. une commune Exar simple ;
3. une carte avec `Prononcer` ;
4. une carte Eshar qui interagit avec l'exil ;
5. une carte Morar modulable ;
6. une carte Velar agressive ;
7. un Vasar prismatique ;
8. un Exar prismatique.

Ces huit cartes servent d'étalon pour le cadre, le wording, le barème et la DA.
